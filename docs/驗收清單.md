# UI 1:1 驗收清單

## Step 4: 驗收清單

請按照以下清單逐項檢查，確保達到 1:1 仿照參考 UI（Travel OS）。

### Layout（佈局）
- [ ] Header 容器寬度為 `max-w-4xl`，左右 padding 為 `px-4` (16px)
- [ ] Header 垂直 padding 為 `py-3` (12px)
- [ ] Main 內容區 padding 為 `px-4 py-4` (16px)
- [ ] 卡片之間間距為 `space-y-4` (16px)
- [ ] 卡片內部 padding 為 `p-4` (16px)
- [ ] 展開後內容區 padding 為 `p-6` (24px)

### Typography（字體）
- [ ] 主標題 (H1) 為 `text-xl` (20px), `font-bold` (700)
- [ ] 副標題為 `text-xs` (12px), `text-slate-500`
- [ ] 日期標籤為 `text-sm` (14px), `font-semibold`
- [ ] 正文為 `text-sm` (14px), `leading-6` (24px)
- [ ] 按鈕文字為 `text-xs` (12px), `font-medium`
- [ ] 所有文字行高為 `leading-6` (24px)

### Colors（顏色）
- [ ] 背景主色為 `bg-white` (#ffffff)
- [ ] 背景次色為 `bg-slate-50` (#f8fafc)
- [ ] 文字主色為 `text-slate-900` (#0f172a)
- [ ] 文字次色為 `text-slate-700` (#334155)
- [ ] 文字輔助為 `text-slate-500` (#64748b)
- [ ] 邊框為 `border-slate-200` (#e2e8f0)
- [ ] 主色調為 `blue-500` (#3b82f6)
- [ ] 連線指示為 `green-500` (#22c55e)
- [ ] 版本標籤為 `bg-yellow-100 text-yellow-700`
- [ ] Hover 狀態為 `hover:bg-slate-100`

### Components（元件）

#### Header
- [ ] Header 有 `border-b border-slate-200`
- [ ] Header 有 `shadow-sm`
- [ ] Header 為 `sticky top-0 z-20`
- [ ] 返回按鈕為 `p-1.5`，hover 為 `hover:bg-slate-100`
- [ ] 連線指示器為 `w-2 h-2 bg-green-500 rounded-full`
- [ ] 版本標籤為 `px-2 py-1 bg-yellow-100 text-yellow-700 text-xs font-medium rounded`

#### Search Bar
- [ ] 搜索欄為 `w-full pl-10 pr-4 py-2.5`
- [ ] 搜索欄圓角為 `rounded-lg` (8px)
- [ ] 搜索欄邊框為 `border border-slate-200`
- [ ] 搜索圖標為 `size={18}`，位置 `left-0 pl-3`
- [ ] Focus 狀態為 `focus:ring-2 focus:ring-blue-500`

#### Date Selector
- [ ] 日期按鈕 padding 為 `px-4 py-2`
- [ ] 日期按鈕圓角為 `rounded-lg` (8px)
- [ ] 選中狀態為 `bg-blue-500 text-white shadow-md`
- [ ] 未選中為 `bg-slate-100 text-slate-700`
- [ ] Hover 為 `hover:bg-slate-200`
- [ ] 按鈕間距為 `gap-2` (8px)

#### Card
- [ ] 卡片背景為 `bg-white`
- [ ] 卡片圓角為 `rounded-lg` (8px)
- [ ] 卡片邊框為 `border border-slate-200`
- [ ] 卡片陰影為 `shadow-sm`
- [ ] Hover 為 `hover:shadow-md`
- [ ] 卡片 padding 為 `p-4` (16px)

#### Bottom Nav
- [ ] 底部導航為 `fixed bottom-0`
- [ ] 底部導航背景為 `bg-white`
- [ ] 底部導航邊框為 `border-t border-slate-200`
- [ ] 圖標大小為 `size={20}`
- [ ] 文字為 `text-xs font-medium`
- [ ] Active 狀態為 `text-blue-500`
- [ ] Active 指示器為 `w-8 h-0.5 bg-blue-500`，位置在按鈕底部中央
- [ ] 按鈕容器為 `relative`（用於定位指示器）

#### Floating Button
- [ ] 浮動按鈕大小為 `w-14 h-14` (56px)
- [ ] 浮動按鈕圓角為 `rounded-full`
- [ ] 浮動按鈕位置為 `fixed bottom-20 right-4`
- [ ] 浮動按鈕背景為 `bg-blue-500`
- [ ] 浮動按鈕陰影為 `shadow-lg`
- [ ] Hover 為 `hover:bg-blue-600`

### Spacing（間距）
- [ ] 使用 Tailwind spacing scale: 2px, 4px, 8px, 12px, 16px, 24px
- [ ] Header gap 為 `gap-2` (8px) 或 `gap-3` (12px)
- [ ] 卡片間距為 `space-y-4` (16px)
- [ ] 內容區 padding 為 `p-6` (24px)

### Interactions（互動）
- [ ] 所有按鈕有 `transition-colors`
- [ ] Hover 狀態為 `hover:bg-slate-100`
- [ ] Focus 狀態為 `focus:ring-2 focus:ring-blue-500`
- [ ] Transition 時間為 `duration-200`

### RWD（響應式）
- [ ] 在 sm breakpoint (≥640px) 時，main padding 為 `sm:px-6`
- [ ] 所有元件在小螢幕上正常顯示
- [ ] 日期選擇器可橫向滾動

### 特殊功能
- [ ] 搜索欄有 "Q " 前綴在 placeholder
- [ ] 日期選擇器選中狀態有 X 按鈕可取消
- [ ] 卡片可展開/摺疊
- [ ] 底部導航 active 狀態正確顯示

## 完成度檢查

完成以上所有項目後，UI 應與參考 Travel OS 設計達到 1:1 匹配。
